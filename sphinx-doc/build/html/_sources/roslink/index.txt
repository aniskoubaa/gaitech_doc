==========================
ROSLink
==========================
 
   
.. toctree::
   :hidden:
   
   ROSLink Overview <roslink-overview.rst>
   ROSLink for Parrot <roslink-parrot.rst>
   ROSLink for Turtlebot <roslink-turtlebot.rst>

ROSLink is a new protocol to integrate Robot Operating System (ROS) enabled-robots with the IoT. The motivation behind ROSLink is the lack of ROS functionality in monitoring and controlling robots through the Internet.

For more details please see: `ROSLink paper`_

.. _ROSLink paper: http://www.springer.com/gp/book/9783319549262
