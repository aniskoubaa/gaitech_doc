===========
Maintenance
===========

Tires
-----

* Make a frequent check if tires are properly filled with air.
* Check nuts of tires are not loosened
* Replace them when top layer is vanished due to heavy friction

Screws
------

* Have a frequent check and make sure they are not loosened
	
Battery
-------

* Make sure battery is not fully discharged. 
* While charging robot always turn main power switch OFF
* Donâ€™t run robot if battery is extremely low
* Frequent check battery run time and replace if necessary 

Internal Components
-------------------

* PC need to be checked and make sure fan properly working
